# 테스트 질의 30개

생성된 스마트폰 로그 데이터를 기반으로 테스트할 수 있는 자연어 질의 목록입니다.

## 📞 통화 관련 질의 (Call Events)

1. 최근 통화한 사람은 누구야?
2. 김철수랑 언제 통화했어?
3. 어제 통화 기록 알려줘
4. 가장 오래 통화한 사람은 누구야?
5. 박민수와 통화한 시간은 몇 분이야?

## 📱 앱 사용 관련 질의 (App Usage Events)

6. 가장 자주 쓴 앱 뭐야?
7. 어제 유튜브 얼마나 봤어?
8. 카카오톡 사용 시간 알려줘
9. 인스타그램 언제 썼어?
10. 가장 많이 사용한 앱 3개 알려줘

## 📅 일정 관련 질의 (Calendar Events)

11. 오늘 일정 뭐 있어?
12. 회의 일정 알려줘
13. 프로젝트 킥오프 회의는 언제야?
14. 다음 주 일정 있어?
15. 회의 장소 어디야?

## 📍 방문 장소 관련 질의 (Visit Events)

16. 어제 어디 갔어?
17. 최근 방문한 카페 어디야?
18. 스타벅스 언제 갔었어?
19. 회사에 몇 시에 도착했어?
20. 주말에 어디 갔었어?

## 📸 콘텐츠 관련 질의 (Content)

21. 최근에 찍은 사진 있어?
22. 어디서 사진 찍었어?
23. 한강공원에서 사진 찍었어?

## 🔗 복합 질의 (Dense Data - 관계가 명확함)

24. 김철수랑 통화하고 카페 갔어?
25. 회의 전에 뭐 했어?
26. 유튜브 보면서 어떤 일정이 있었어?
27. 집에서 나가기 전에 누구랑 통화했어?

## 🔮 Sparse Data 질의 (Link Prediction 필요)

**이 질의들은 관계가 누락되어 있어서 Link Prediction이 작동해야 해요:**

28. 김철수랑 통화하고 나서 들른 카페 어디였지?
   - visitedAfter 관계 누락

29. 프로젝트 킥오프 회의에 누가 참석했어?
   - attendedBy 관계 누락

30. 투썸플레이스에서 누구 만났어?
   - metDuring 관계 누락

---

## 🎯 테스트 방법

### Dense Data 테스트 (1-27번)
- Link Prediction **끄기**
- 기본 SPARQL로 답변 가능
- 빠른 응답

### Sparse Data 테스트 (28-30번)
- Link Prediction **켜기** (체크박스)
- 누락된 관계 예측
- "[예측]" 표시와 함께 답변

## 📊 예상 결과 예시

**질의**: "최근 통화한 사람은 누구야?"
**답변**: "김철수님, 최대한님과 최근 통화하셨습니다."

**질의**: "김철수랑 통화하고 나서 들른 카페 어디였지?" (Link Prediction 켜기)
**답변**: "[예측] 통화 후 스타벅스 역삼점을 방문하셨을 가능성이 높습니다."
